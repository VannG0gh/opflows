Webhook Information

How to set up Jira webhooks:

  1. Go to your Jira project settings
  2. Navigate to Automation → Rules
  3. Create a new rule with Webhook received trigger
  4. Copy the webhook URL and paste it above
  5. Configure your automation actions (create issues, send notifications, etc.)
