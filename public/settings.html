Jira API Information

How to set up Jira API access:

  1. Go to https://id.atlassian.com/manage/api-tokens and generate an API token
  2. Enter your Jira domain (the part before '.atlassian.net' in your Jira URL)
  3. Enter the email address associated with your Jira account
  4. Enter your project key (found in Jira project settings)
  5. The system will automatically create epics, tasks, and subtasks using the Jira REST API
